# Калькулятор

Простой консольный калькулятор на Python, который умеет:
- Сложение
- Вычитание
- Умножение
- Деление

## Как использовать

1. Клонируйте репозиторий:
```bash
git clone https://github.com/ALBOTAM/Calculator.git
```
2. Перейдите в папку проекта:
   ```cd Calculator```
3. Запустите калькулятор:
  ```python calculator.py```

4. simple_calculator/
   │
   ├─ calculator.py      # Главный файл с кодом калькулятора
   ├─ README.md          # Описание проекта и инструкция по запуску
   ├─ requirements.txt   # Файл зависимостей (пустой, так как используется стандартная библиотека)
   └─ .gitignore         # Файл для игнорирования временных/ненужных файлов
